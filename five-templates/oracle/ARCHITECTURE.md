# Oracle Template - Architecture

A minimal, modular layout for oracle patterns in Five DSL.

## Layout

```
oracle/
├── five.toml
├── src/
│   ├── types/
│   ├── oracle/
│   └── main.v
└── PROJECT_SUMMARY.md
```

## Design Goals

- Keep types isolated for reuse
- Keep core logic in focused modules
- Provide a simple main entry point
